# Rust Language

## Installation

### Windows

- Download Microsoft C++ build tools.  
- Download rustup tool

### Linux

- Install `build essential` package.
```
sudo apt install build-essential
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"
```
